# LUMI Air Pro — Premium Product Landing Page Demo

A modern premium-style product landing page demo inspired by Apple's product presentation style.  
Built using pure HTML, Tailwind CSS CDN, custom CSS, and Vanilla JavaScript.

This project is designed as a portfolio/demo template to showcase frontend UI/UX, animation, responsive layouts, and premium landing page design concepts for freelance clients and presentations.

---

## Preview

LUMI Air Pro is a fictional premium wireless headphone product used for demonstration purposes only.

Features included in this demo:

- Apple-inspired premium UI
- Smooth scroll & reveal animations
- Responsive layout
- Bilingual-ready structure (TH / EN)
- Modern glassmorphism effects
- Premium CTA sections
- Product showcase hero section
- Interactive hover effects
- Clean typography and spacing system

---

## Technologies Used

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Custom CSS
